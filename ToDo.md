## このリポジトリで実現したいこと
このリポジトリはプログラミング初心者が初めて1人で Web アプリを作成して、AWS 上公開することを実現できるプロジェクトのテンプレートを作成したいです。
このテンプレートを利用する初心者は、以下の技術の基礎を一通りインプットした上でこの演習課題に臨みます。
- TypeScript
- React
- Node.js
- Vite
- Vitest
- Docker/Docker Compose

このリポジトリはローカルで Docker Compose で動かしつつ、AWS CDK で ECR にデプロイしたコンテナをECSで動かすことを実現したいです。
コンテナはフロントエンド、バックエンド、DBの3つを想定しています。
DBは MySQL を採用、CDKで RDS インスタンスを立ち上げた上で ECS 上に動かすという内容にしたい。
初心者にとってはいきなり実装しろと言われても何から手をつけたらいいかがわからないので、
GET・POST・PUTのサンプルを事前に用意しておきたいです。
フロントエンドのスタイルは Tailwind CSS を採用したいです。
また、インプットはしていないですが、バックエンドのフレームワークはHonoを採用したいです。（初心者がリファレンスを読みながら実装を進める想定）

README.md には以下の手順を明記しておきたいです。
1. テンプレートから自前のリポジトリを作成する手順
1. 自前のテンプレートをローカルに落とす手順
1. ローカルでアプリを動かす手順
1. アプリを AWS にデプロイする手順
